🎓 نظام إدارة التعليم "الحصة" — منصة تعليمية متكاملة للمدارس والدروس الخصوصية
💡 الرؤية:
منصة تعليمية حديثة تمكّن المعلم من إنشاء وإدارة الفصول الدراسية الرقمية بسهولة، مع دعم كامل للتفاعل، الواجبات، المسابقات، إدارة الطلاب، والتقييمات، ومبنية بتقنيات حديثة توفر تجربة سلسة وفعالة لجميع الأطراف (المعلم، الطالب، المساعد، المشرف).

🧱 التقنيات المستخدمة:
الواجهة: HTML + CSS + JavaScript (تصميم عصري وسهل الاستخدام)

الخلفية: Flask (Python) + MySQL

الوظائف التفاعلية الفورية: Firebase (لتسجيل الدخول، الإشعارات، والدردشة)

التصميم: Responsive UI مع أنيميشن بسيطة، ألوان مرنة حسب الفصل، واجهة داكنة/فاتحة

👤 أنواع المستخدمين:
الطالب

المعلم

مساعد المعلم

المدير (Admin)

📚 وظائف المعلم:
إنشاء فصل بكود فريد + اسم + سنة دراسية + مادة + صورة + لون + مجاني/مدفوع (سعر + مدة)

إضافة محتوى متنوع داخل الفصل (ملف، صورة، صوت، فيديو، نص)

إنشاء:

واجبات

كويزات (مع جدولة)

اختبارات

مسابقات

تقييم أداء الطلاب بالنقاط

إدارة الطلاب (تفعيل/تعطيل/حذف/إرسال إشعار)

تعيين مساعد

إدارة شات الفصل (إضافة/إزالة طلاب، حذف رسائل)

مشاهدة الطلاب النشطين حالياً

🧑‍🎓 وظائف الطالب:
الانضمام لفصل باستخدام كود

في حالة الدفع: يتم تحويله لبوابة الدفع

إن كان مجاني: يتم الدخول مباشرة

التفاعل مع المحتوى

حل الواجبات، الكويزات، المسابقات

إرسال أسئلة للمعلم

التفاعل في الشات (إن وُفّر)

🧑‍🏫 وظائف مساعد المعلم:
التسجيل كمساعد + إدخال كود الفصل

يجب موافقة المعلم ليتم تعيينه

يمكنه:

إدارة الشات

تقييم الواجبات

متابعة المسابقات والكويزات

🛠️ وظائف المدير (Admin):
إدارة المستخدمين (تفعيل/تعطيل معلمين)

تحديد فترة تجريبية عند التسجيل الأول للمعلمين (يتم إتاحة كل الخصائص خلالها)

التحكم الكامل في:

الفصول

أداء الطلاب (حسب نقاط الكويزات والواجبات)

الاشتراكات (إنشاء/تعديل باقات مع المميزات والسعر)

عروض على الباقات/الكتب/المذكرات

الإشعارات (إرسال حسب الدور أو لمستخدم محدد بالاسم أو رقم الهاتف)

الكورسات والمذكرات للشراء

إعدادات الموقع (الألوان، البنرات، الخط، الإعلانات)

🔒 التسجيل وتحديث الملف الشخصي:
يتم التسجيل باستخدام:

الاسم

رقم الهاتف (عبر Firebase)

الدور (طالب/معلم/مساعد)

صفحة ملف شخصي لكل مستخدم:

تعديل الاسم، الصورة، الاهتمامات، العنوان

رقم الهاتف لا يمكن تغييره (يمكن إضافة رقم بديل)

💬 الشات والمراسلة:
يتم دمج Firebase Chat داخل كل فصل (اسم الشات = اسم الفصل)

يظهر فقط للطلاب المضافين من المعلم أو المساعد

المعلم/المساعد يمكنهم حذف طالب من الشات

الطالب المحذوف لا يستطيع رؤية الشات بعد الحذف

📈 التحليلات والاشتراكات:
3 باقات اشتراك (يتم تحديد مميزاتها من قبل الأدمن):

باقة 1: فصل واحد فقط، بدون شات

باقة 2: فصول متعددة، شات، لا مساعد

باقة 3 (كاملة): كل الخصائص + صفحة تحليلات متقدمة داخل كل فصل + إمكانية تعيين مساعد

أثناء الفترة التجريبية: يتم تفعيل جميع الخصائص تلقائياً

🔔 الإشعارات:
باستخدام Firebase Cloud Messaging

يمكن للأدمن إرسال إشعارات لـ:

جميع الطلاب أو المعلمين أو المساعدين

مستخدم معين (باستخدام الاسم أو رقم الهاتف)

📦 الخصائص القابلة للتوسعة لاحقاً:
نظام تقييم شامل للطلاب على مستوى المنصة

نظام شارات أو أوسمة

بث مباشر أو فصول مباشرة

سوق لبيع المذكرات بين المعلمين والطلاب
